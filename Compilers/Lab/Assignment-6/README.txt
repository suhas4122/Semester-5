FOLDER STRUCTURE 

test-inputs: Contains 5 input .c test files 
test-outputs: Contains the three adress codes and test outputs 
bin: Contains the binaries resulted from compilation of input test files

COMPILE INSTRUCTIONS 

make: to generate the yatcc binary 
make run: to compile all the input test files and generate quads, three adress codes and binaries
make clean: to delete all the temporary and output files before compiling again 

RUNNING THE TEST FILES 

Open the folder bin and run the following command on terminal 
./test<x> : to run the xth test file 
Eg. running ./test1 runs the first test file
